The file <prices.csv> contains the items associated prices in terms of the specific store.

First column represents the item title.
The further columns represent the store associated prices.

3 Stores were investigated, with following store_id - name relation:
	store_0 : Amazon
	store_1 : Spielemax
	store_2 : Toys for fun



The file <results.csv> contains the items associated metrics of the price analysis.

The first column represents the item title again.

The second one represents the ratio between the price of store_0 and store_2 (store_1 is ignored because it's not relevant).
The third one represents the absolute price difference between store_0 and store_2.
